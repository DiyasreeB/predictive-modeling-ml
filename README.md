Predictive Modeling Using Machine Learning

Project Objective
The objective of this project is to build a predictive machine learning model that can analyze input data and accurately predict outcomes using supervised learning algorithms. The project focuses on comparing multiple models and evaluating their performance using standard metrics.

Dataset Used
This project uses the Titanic Dataset, which contains information about passengers such as age, gender, ticket class, and survival status.
Dataset Link:
https://raw.githubusercontent.com/datasciencedojo/datasets/master/titanic.csv

Machine Learning Models Used
The following supervised learning models were implemented:
-Decision Tree Classifier
-Random Forest Classifier
-Logistic Regression (optional baseline model)

Model Comparison
Model	                Accuracy Score
Decision Tree     	~  0.78 – 0.82
Logistic Regression	~  0.80 – 0.85
Random Forest     	~  0.83 – 0.88
Note: Accuracy may vary depending on train-test split and random state.

Evaluation Metrics Used
-Accuracy Score
-Confusion Matrix
-ROC Curve and AUC Score

Result Summary
-Random Forest Classifier performed the best among all models due to its ability to reduce overfitting and handle complex data patterns.
-Decision Tree provided good interpretability but slightly lower accuracy.
-Logistic Regression served as a strong baseline model.
Final Conclusion: Random Forest is the most suitable model for this dataset.
