Predictive Modeling Using Machine Learning

Objective
Build machine learning models to predict outcomes using supervised learning.

- Random Forest Classifier

Steps
1. Data cleaning
2. Feature selection
3. Model training
4. Model evaluation
5. Visualization (Confusion Matrix, ROC Curve)

Dataset
Titanic dataset (Kaggle)

Result
Random Forest performed best with highest accuracy.
